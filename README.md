# Cedrick Rojas — Portfolio

This repository contains a static HTML portfolio page: `cedrick-rojas-portfolio_2.html`.

Deployment options:

- GitHub Pages (recommended):
  1. Ensure the repository is public (or use a personal/organization Pages configuration).
  2. In repository Settings → Pages, set the source to the `main` branch and the root folder.
  3. The site will be available at the repo root, and `index.html` will load the portfolio without showing `cedrick-rojas-portfolio_2.html` in the URL.

- Vercel:
  1. Connect your GitHub account to Vercel and import this repository.
  2. Use default settings for a static site — Vercel will deploy automatically on push.
  3. Open the root URL (`/`) rather than the source filename.

Notes:
- `index.html` loads `cedrick-rojas-portfolio_2.html` behind the scenes so the deployed URL stays clean.
- Images are located in `picture/`.
